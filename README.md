GOAL:
An AI-powered Social Media Trend Forecaster & Short Video Script Generator.
The app predicts trending topics and automatically generates short-form video scripts (Reels, Shorts, TikTok).

TECH STACK:
- Frontend: Modern responsive UI (dark theme, clean SaaS style)
- Backend: Node.js or serverless functions
- Database: Store users, saved scripts, trend history
- AI: OpenAI API integration for trend prediction and script generation
- Optional: Integrate Google Trends API or Twitter/X API (mock if needed)
