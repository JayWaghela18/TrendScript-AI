# TrendScript-AI

**TrendScript-AI** is an AI-powered Social Media Trend Forecaster and Short Video Script Generator. It helps content creators, marketers, and social media enthusiasts discover trending topics and automatically generate high-engagement short-form video scripts for platforms like Instagram Reels, YouTube Shorts, and TikTok.
